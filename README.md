# 🌎 Observatório Digital de Camaçari

Um portal web desenvolvido como Produto Final da disciplina de Informática para Internet, com o objetivo de reunir, organizar e divulgar projetos sociais, patrimônios culturais e locais de interesse do município de Camaçari por meio de um mapa interativo e uma plataforma digital acessível.

---

## 📖 Sobre o Projeto

O Observatório Digital de Camaçari foi criado para facilitar o acesso às informações sobre iniciativas sociais, culturais e comunitárias do município.

A plataforma reúne dados pesquisados pela equipe e os apresenta de forma interativa, permitindo que qualquer pessoa conheça projetos, localidades importantes e ações voltadas ao desenvolvimento social e ao Bem Viver.

O sistema foi desenvolvido utilizando tecnologias da Web e boas práticas de desenvolvimento colaborativo com Git e GitHub.

---

## 🎯 Objetivos

- Divulgar projetos sociais existentes em Camaçari.
- Valorizar espaços culturais e patrimônios locais.
- Facilitar a localização das iniciativas através de um mapa interativo.
- Incentivar a participação da comunidade.
- Promover o acesso democrático às informações.

## 🌍 Mapa Interativo

O mapa foi desenvolvido utilizando a biblioteca Leaflet.js integrada ao OpenStreetMap.

Cada marcador representa um projeto ou local relevante pesquisado pela equipe.

Os usuários podem:

- visualizar projetos;
- pesquisar localidades;
- aplicar filtros por categoria;
- acessar informações detalhadas através de popups.

## 📌 Metodologia

O projeto foi desenvolvido utilizando controle de versão com Git.

Cada funcionalidade foi implementada em uma branch específica e posteriormente integrada ao projeto principal por meio de Pull Requests.

Fluxo utilizado:

```
feature
    ↓

develop
    ↓

main
```
